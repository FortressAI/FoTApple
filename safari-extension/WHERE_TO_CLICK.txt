❓ COMMON QUESTIONS
════════════════════════════════════════════════════════════════════════════════

Q: I don't see "QFOT Wallet (macOS)" in the dropdown
A: Try "FoT Wallet (macOS)" - same thing, different name

Q: The dropdown only shows iOS options
A: Product → Scheme → Manage Schemes → Enable macOS schemes

Q: Build still fails
A: Product → Clean Build Folder (⌘⇧K), then try again

Q: I don't see the extension in Safari
A: Make sure the app is RUNNING in Xcode (see "Running..." status)

Q: Do I need to deploy this?
A: NO! It runs locally. Blockchain is already deployed.

Q: Do I need App Store?
A: NO! Not for personal use. Only if you want public distribution.
