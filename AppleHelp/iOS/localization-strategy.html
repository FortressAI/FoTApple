Translation Quality Standards

Professional Translation Required

No Machine Translation for Critical Content: - ‚ùå Google Translate, DeepL, ChatGPT alone - ‚úÖ Professional medical/legal translators only - ‚úÖ Native speaker review - ‚úÖ Domain expert validation

Why: Medical and legal terminology must be precise. Machine translation can introduce dangerous errors (e.g., confusing "contraindicated" with "recommended").

Translation Workflow

1. Source Preparation (Week 1)

- Extract all UI strings to .strings files (iOS)
- Export database content (drug names, symptoms, legal terms)
- Prepare glossary of domain terms
- Create translation memory (TM) for consistency

2. Professional Translation (Weeks 2-6)

- Hire certified medical/legal translators
- Translate UI strings
- Translate domain content
- Translate documentation
- Create language-specific glossaries

3. Technical Review (Week 7)

- Import translations to app
- Test UI layout (text expansion/contraction)
- Verify formatting (dates, numbers, currency)
- Check right-to-left languages (Arabic)
- Test on actual devices

4. Domain Expert Review (Week 8)

- Medical translations reviewed by native-speaking physician
- Legal translations reviewed by native-speaking attorney
- Education translations reviewed by native-speaking teacher
- Corrections made, retested

5. Linguistic QA (Week 9)

- Native speakers test complete workflows
- Identify awkward phrasing, cultural issues
- Verify terminology consistency
- Check for truncation, overlaps

6. Final Approval & Ship (Week 10)

- Client (FoT Apple) approves translations
- Submit to App Store with localized metadata
- Launch marketing campaign in target languages

Right-to-Left (RTL) Support

Arabic requires UI mirroring:

// SwiftUI handles RTL automatically, but test thoroughly:
HStack {
    Image(systemName: "chevron.right") // Flips to left in RTL
    Text("next_button")
}
.environment(\.layoutDirection, .rightToLeft) // For testing

What to mirror: - Navigation (back buttons, etc.) - Reading order (left-to-right becomes right-to-left) - Icon orientation

What NOT to mirror: - Medical diagrams (heart is on left side regardless of language) - Numbers, graphs - Brand logos
