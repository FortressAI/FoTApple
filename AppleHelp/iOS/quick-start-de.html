Für Kliniker

FoT Clinician - Medizinische Entscheidungsunterstützung

Was Sie Benötigen: - iPhone (iOS 17+), Mac (macOS 14+), oder Apple Watch (watchOS 10+) - 5 Minuten für die Einrichtung - Internetverbindung (für initiales Setup und Arzneimitteldatenbank)

Schritt 1: Installation (2 Minuten)

TestFlight Beta (Jetzt Verfügbar): 1. Senden Sie eine E-Mail an beta@fortressai.com mit: - Ihrer medizinischen Fachrichtung - NPI-Nummer (zur Verifizierung) - Gerätepräferenz (iPhone/Mac/Watch) 2. Erhalten Sie TestFlight-Einladung innerhalb 24-48 Stunden 3. Installieren Sie die App über den TestFlight-Link

App Store (Geplant Q2 2026): - Suchen Sie "FoT Clinician" - Herunterladen und installieren - 30-tägige kostenlose Testversion inklusive

Schritt 2: Richten Sie Ihr Konto Ein (1 Minute)

  1. Starten Sie FoT Clinician
  2. Erstellen Sie ein Konto mit Face ID oder Touch ID
  3. Konfigurieren Sie Präferenzen:
  4. Einheiten (metrisch oder imperial)
  5. Klinische Leitlinien (ACC/AHA, GINA, etc.)
  6. Benachrichtigungseinstellungen
  7. Schließen Sie das kurze Tutorial ab (überspringen, wenn erfahren)

Schritt 3: Probieren Sie Schlüsselfunktionen (2 Minuten)

Fügen Sie Ihren Ersten Patienten Hinzu:

1. Tippen Sie auf "+" Button
2. Geben Sie Patientendetails ein:
   - Name (oder verwenden Sie "Max Mustermann" für Demo)
   - Alter: 68
   - Geschlecht: Männlich
3. Fügen Sie Allergien hinzu: Penicillin
4. Fügen Sie aktuelle Medikamente hinzu:
   - Metformin 1000mg PO BID
   - Lisinopril 20mg PO täglich
5. Speichern Sie den Patienten

Prüfen Sie Arzneimittelwechselwirkungen:

1. Wählen Sie Patient "Max Mustermann"
2. Tippen Sie auf "Konsultation"-Tab
3. Scrollen Sie zum "Medikamente"-Bereich
4. Tippen Sie auf "Medikament Hinzufügen"
5. Suchen Sie "Aspirin"
6. Wählen Sie "Aspirin 81mg"
7. Tippen Sie auf "Wechselwirkungen Prüfen"
8. ⚠️ Sehen Sie Wechselwirkungswarnung mit Metformin
9. Überprüfen Sie Mechanismus und Empfehlungen

Generieren Sie SOAP-Notiz:

1. In der Konsultation, fügen Sie hinzu:
   - Hauptbeschwerde: "Brustschmerzen"
   - Vitalzeichen: RR 145/92, HF 98, AF 18
2. Scrollen Sie zum "SOAP-Notiz"-Bereich
3. Tippen Sie auf "Generieren"
4. Überprüfen Sie automatisch generierte professionelle Notiz
5. Tippen Sie auf "Exportieren" zum Teilen (PDF)

Zeigen Sie Kryptografischen Beweis An:

1. Am Ende der SOAP-Notiz
2. Sehen Sie "Blockchain-Beweis: 01JD7X8K..."
3. Tippen Sie, um vollständigen Beweis zu sehen
4. Sehen Sie Merkle-Root, Ed25519-Signatur
5. Unabhängig verifizierbar auf SafeAICoin

Für Anwälte

FoT Legal US - Kanzleiverwaltung

Was Sie Benötigen: - iPhone (iOS 17+), iPad (iOS 17+), oder Mac (macOS 14+) - 5 Minuten für die Einrichtung - Ihre Anwaltszulassungsinformationen

Schritt 1: Installation (2 Minuten)

TestFlight Beta (Jetzt Verfügbar): 1. Senden Sie eine E-Mail an beta@fortressai.com mit: - Staatliche Anwaltszulassung und Jahr - Anwaltsnummer (zur Verifizierung) - Praxisbereiche 2. Erhalten Sie TestFlight-Einladung 3. Installieren Sie die App

Schritt 2: Richten Sie Ihr Konto Ein (1 Minute)

  1. Starten Sie FoT Legal US
  2. Erstellen Sie ein Konto mit Face ID oder Touch ID
  3. Konfigurieren Sie Präferenzen:
  4. Hauptgerichtsbarkeit (Bundes-/Landesebene)
  5. Praxisbereiche (Zivil, Straf, etc.)
  6. Gerichtsregeln (FRCP, lokale Regeln)
  7. Schließen Sie Tutorial ab

Schritt 3: Probieren Sie Schlüsselfunktionen (2 Minuten)

Erstellen Sie Ihren Ersten Fall:

1. Tippen Sie auf "+" Button
2. Geben Sie Falldetails ein:
   - Name: Smith gegen Johnson Corp.
   - Typ: Zivil - Vertragsbruch
   - Gericht: US-Bezirksgericht, ND Kalifornien
   - Eingereicht: [Heutiges Datum]
3. Fügen Sie Parteien hinzu:
   - Mandant: Sarah Smith
   - Gegenseite: Johnson Corporation
4. Speichern Sie den Fall

Automatische Fristenberechnung:

1. Öffnen Sie Fall "Smith gegen Johnson"
2. Sehen Sie "Fristen"-Tab
3. Sehen Sie automatisch berechnete Fristen:
   - Antwort fällig: 21 Tage (FRCP 12(a)(1)(A)(i))
   - Erstmitteilungen: [Datum] (FRCP 26(a)(1)(C))
   - Discovery-Konferenz: [Datum] (FRCP 26(f))
4. Jede Frist enthält Regelverweis
5. Erinnerungen automatisch gesetzt

Generieren Sie Bluebook-Zitat:

1. Tippen Sie auf "Recherche"-Tab
2. Suchen Sie "NFIB v Sebelius"
3. Wählen Sie Fall aus Ergebnissen
4. Tippen Sie auf "Zitat Kopieren"
5. Zitat im Bluebook-Format kopiert:
   Nat'l Fed'n of Indep. Bus. v. Sebelius, 567 U.S. 519 (2012).

Verfassungsanalyse:

1. Tippen Sie auf "Analyse"-Tab
2. Geben Sie Frage ein: "Kann der Kongress Krankenversicherung verlangen?"
3. System liefert:
   - Relevante Verfassungsbestimmungen
   - Supreme Court Präzedenzfall (NFIB v. Sebelius)
   - Analyse Commerce Clause vs. Steuergewalt
   - Empfohlene Strategie
4. Überprüfen Sie Blockchain-Beweis der Analyse

Für Entwickler

Bauen und Beitragen

Was Sie Benötigen: - Mac mit macOS 14+ (Sonoma) - Xcode 15.0+ - Git - 10 Minuten für initiales Setup

Schritt 1: Repository Klonen (1 Minute)

# Hauptrepository klonen
git clone https://github.com/FortressAI/FoTApple.git
cd FoTApple

# Struktur ansehen
ls -la

Repository-Struktur:

FoTApple/
├── apps/                    # 3 Anwendungen (Clinician, Legal, Education)
├── packages/                # 6 wiederverwendbare Swift-Pakete
├── scripts/                 # Automatisierungsskripte
└── docs/                    # Dokumentation

Schritt 2: Automatisiertes Setup (5 Minuten)

Für Clinician iOS App:

cd apps/ClinicianApp/iOS
./setup.sh

Was setup.sh Macht: 1. ✅ Überprüft Systemabhängigkeiten (Xcode, xcodegen) 2. ✅ Installiert xcodegen falls fehlend 3. ✅ Generiert Xcode-Projektdatei 4. ✅ Löst Swift-Paket-Abhängigkeiten auf 5. ✅ Erstellt Initialkonfiguration 6. ✅ Führt Infrastrukturvalidierung aus

Dauer: ~5 Minuten beim ersten Mal

Schritt 3: Bauen und Testen (4 Minuten)

In Xcode Öffnen:

# Nach setup.sh
open ClinicianApp.xcodeproj

Oder Build-Skript Verwenden:

# Tests ausführen
./build.sh test

# Für iPhone bauen
./build.sh run

# Für Mac bauen
./build.sh run-mac

# Alle Optionen sehen
./build.sh help

Erste Tests:

# Vollständige Suite ausführen (54+ Tests)
./build.sh test

# Spezifische Tests ausführen
xcodebuild test \
  -scheme ClinicianApp \
  -destination 'platform=iOS Simulator,name=iPhone 15'

Funktioniert es offline?

Offline-Funktionalität:

✅ Funktioniert Vollständig Offline: - Bestehende Patienten/Fälle/Schüler ansehen - Neue Datensätze hinzufügen - Lokale Reasoning-Berechnungen (AKG GNN) - SOAP-Notizen, Fristenberechnungen, Aufgaben

⚠️ Benötigt Verbindung Für: - Arzneimittelwechselwirkungsprüfungen (RxNav API) - Rechtsprechungsrecherche (Rechtsdatenbanken) - Blockchain-Attestierung (Beweis-Einreichung) - Synchronisation zwischen Geräten

Lokale Datenspeicherung: - SQLite + GRDB für lokale Persistenz - Verschlüsselt mit Secure Enclave verwalteten Schlüsseln - Synchronisierbar über iCloud wenn verbunden
