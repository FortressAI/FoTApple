Overview

Field of Truth is not three separate apps - it's a unified sensor-capture and truth-attestation platform that serves individuals across all life contexts. Every app shares the same core infrastructure for capturing reality with device sensors (camera, GPS, accelerometer, microphone, etc.) and creating cryptographically-attested receipts.

Unified Technical Architecture

Sensor Capture Layer (Shared)

protocol SensorCaptureEngine {
    func captureIncident() async -> IncidentReceipt
    func getAvailableSensors() -> [SensorType]
    func requestPermissions() async -> Bool
}

enum SensorType {
    case camera, gps, accelerometer, gyroscope,
         magnetometer, barometer, lightSensor,
         microphone, bluetooth, wifi
}

Receipt Generation (Shared)

struct UniversalReceipt {
    let timestamp: Date
    let location: CLLocation?
    let sensorData: [SensorType: Data]
    let cryptographicProof: ReceiptBundle
    let domain: Domain  // legal, education, health
    let sharePermissions: [SharePermission]
}
