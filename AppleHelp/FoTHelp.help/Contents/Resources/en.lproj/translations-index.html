Technical Implementation

File Naming Convention

All translated pages follow the pattern: PageName-{language_code}.md

Examples: - English: Home.md - Spanish: Home-es.md - Chinese: Home-zh.md - French: Home-fr.md - German: Home-de.md - Portuguese (Brazil): Home-pt-BR.md - Japanese: Home-ja.md

Language Detection

Translations are automatically detected and offered based on: 1. Browser language settings 2. IP geolocation 3. User preference (stored in local settings)

Fallback Behavior

If a translation is not available: 1. Display English version with "Translation in Progress" notice 2. Offer to contribute translation via GitHub 3. Provide machine translation option (marked as "Auto-Translated")

Translation Guidelines

For Medical Content: - Use region-specific medical terminology - Maintain clinical accuracy - Preserve drug names (INN preferred) - Include local regulatory equivalents

For Legal Content: - Adapt to local legal systems where applicable - Note US-specific references - Provide international equivalents - Maintain citation accuracy

For Technical Content: - Keep code examples in English - Translate comments and documentation - Preserve technical accuracy - Test all command examples
