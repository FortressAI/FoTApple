üì∏ How to Capture Screenshots

Manual Method (Recommended for Polish)

iPhone: 1. Set up simulator: iPhone 17 Pro (iOS 26.1) 2. Run app in simulator 3. Navigate to desired screen 4. Press Cmd + S to save screenshot 5. Screenshots save to Desktop by default

macOS: 1. Run app on Mac 2. Navigate to desired screen 3. Press Cmd + Shift + 4 then Space to capture window 4. Or use Cmd + Shift + 3 for full screen

Apple Watch: 1. Run on Watch simulator 2. Navigate to desired screen 3. Press Cmd + S

Automated Method (For Bulk Capture)

Create a script scripts/capture_screenshots.sh:

#!/bin/bash

# Capture screenshots for all apps
# Usage: ./scripts/capture_screenshots.sh

SCREENSHOT_DIR="FoTApple.wiki/screenshots/app-personas"
mkdir -p "$SCREENSHOT_DIR"

echo "üì∏ Capturing Clinician App screenshots..."
# Launch app, navigate, capture
xcrun simctl io booted screenshot "$SCREENSHOT_DIR/clinician-01-patient-list.png"

echo "üì∏ Capturing Personal Health App screenshots..."
# Repeat for each app and screen

echo "‚úÖ All screenshots captured to $SCREENSHOT_DIR"

XCUITest Automated Screenshots

For professional App Store screenshots, use XCUITest:

// In UITests target
func testCaptureScreenshots() {
    let app = XCUIApplication()
    app.launch()

    // Navigate to each screen and capture
    app.tabBars.buttons["Patients"].tap()
    takeScreenshot(named: "01-patient-list")

    app.tables.cells.firstMatch.tap()
    takeScreenshot(named: "02-patient-detail")

    // Repeat for all screens
}

func takeScreenshot(named name: String) {
    let screenshot = XCUIScreen.main.screenshot()
    let attachment = XCTAttachment(screenshot: screenshot)
    attachment.name = name
    attachment.lifetime = .keepAlways
    add(attachment)
}

üé® Screenshot Guidelines

Technical Requirements

  • Resolution: 1290√ó2796 for iPhone 17 Pro (3x scale)
  • Format: PNG with transparency where applicable
  • Color Space: sRGB
  • File Size: < 5MB per screenshot

Composition Tips

  1. Show real data (not "Lorem ipsum" or "Test User")
  2. Highlight key features in each screenshot
  3. Use consistent lighting (Appearance: Light or Dark, be consistent)
  4. Show glass effects prominently
  5. Include status bar (shows time, signal, battery)
  6. No personal info (use fictitious names, addresses)
  7. Professional aesthetics (clean, polished, no debug info)
