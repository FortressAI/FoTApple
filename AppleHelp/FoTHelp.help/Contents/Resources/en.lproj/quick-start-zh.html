医疗专业人士

FoT Clinician - 医疗决策支持

您需要的： - iPhone (iOS 17+)、Mac (macOS 14+) 或 Apple Watch (watchOS 10+) - 5分钟设置时间 - 互联网连接（用于初始设置和药物数据库）

步骤1：安装（2分钟）

TestFlight测试版（现已推出）： 1. 发送电子邮件至 beta@fortressai.com，提供： - 医学专业 - NPI号码（用于验证） - 设备偏好（iPhone/Mac/Watch） 2. 在24-48小时内收到TestFlight邀请 3. 从TestFlight链接安装应用

App Store（2026年第二季度推出）： - 搜索"FoT Clinician" - 下载并安装 - 包含30天免费试用

步骤2：设置您的账户（1分钟）

  1. 启动 FoT Clinician
  2. 使用 Face ID 或 Touch ID 创建账户
  3. 配置偏好设置：
  4. 单位（公制或英制）
  5. 临床指南（ACC/AHA、GINA等）
  6. 通知设置
  7. 完成简短教程（有经验可跳过）

步骤3：试用关键功能（2分钟）

添加您的第一位患者：

1. 点击"+"按钮
2. 输入患者详情：
   - 姓名（或使用"张三"作为演示）
   - 年龄：68
   - 性别：男性
3. 添加过敏：青霉素
4. 添加当前药物：
   - 二甲双胍 1000mg 口服 每日两次
   - 赖诺普利 20mg 口服 每日一次
5. 保存患者

检查药物相互作用：

1. 选择患者"张三"
2. 点击"就诊"标签
3. 滚动到"药物"部分
4. 点击"添加药物"
5. 搜索"阿司匹林"
6. 选择"阿司匹林 81mg"
7. 点击"检查相互作用"
8. ⚠️ 查看与二甲双胍的相互作用警告
9. 查看机制和建议

生成SOAP笔记：

1. 在就诊中，添加：
   - 主诉："胸痛"
   - 生命体征：血压 145/92，心率 98，呼吸频率 18
2. 滚动到"SOAP笔记"部分
3. 点击"生成"
4. 查看自动生成的专业笔记
5. 点击"导出"分享（PDF）

查看加密证明：

1. 在SOAP笔记底部
2. 查看"区块链证明：01JD7X8K..."
3. 点击查看完整证明
4. 查看Merkle根、Ed25519签名
5. 可在SafeAICoin上独立验证

您需要的： - iPhone (iOS 17+)、iPad (iOS 17+) 或 Mac (macOS 14+) - 5分钟设置时间 - 您的律师执业信息

步骤1：安装（2分钟）

TestFlight测试版（现已推出）： 1. 发送电子邮件至 beta@fortressai.com，提供： - 州律师执业和年份 - 律师编号（用于验证） - 执业领域 2. 收到TestFlight邀请 3. 安装应用

步骤2：设置您的账户（1分钟）

  1. 启动 FoT Legal US
  2. 使用 Face ID 或 Touch ID 创建账户
  3. 配置偏好设置：
  4. 主要管辖区（联邦/州）
  5. 执业领域（民事、刑事等）
  6. 法院规则（FRCP、地方规则）
  7. 完成教程

步骤3：试用关键功能（2分钟）

创建您的第一个案件：

1. 点击"+"按钮
2. 输入案件详情：
   - 名称：Smith v. Johnson Corp.
   - 类型：民事 - 违约
   - 法院：美国地区法院，加州北区
   - 提交日期：[今天的日期]
3. 添加当事人：
   - 客户：Sarah Smith
   - 对方：Johnson Corporation
4. 保存案件

自动期限计算：

1. 打开案件"Smith v. Johnson"
2. 查看"期限"标签
3. 查看自动计算的期限：
   - 答辩到期：21天（FRCP 12(a)(1)(A)(i)）
   - 初始披露：[日期]（FRCP 26(a)(1)(C)）
   - 证据开示会议：[日期]（FRCP 26(f)）
4. 每个期限都包含规则引用
5. 自动设置提醒

生成Bluebook引文：

1. 点击"研究"标签
2. 搜索"NFIB v Sebelius"
3. 从结果中选择案例
4. 点击"复制引文"
5. 引文以Bluebook格式复制：
   Nat'l Fed'n of Indep. Bus. v. Sebelius, 567 U.S. 519 (2012).

宪法分析：

1. 点击"分析"标签
2. 输入问题："国会能否要求健康保险？"
3. 系统提供：
   - 相关宪法条款
   - 最高法院先例（NFIB v. Sebelius）
   - 商业条款与征税权的分析
   - 推荐策略
4. 查看分析的区块链证明

开发人员

构建和贡献

您需要的： - 运行 macOS 14+ (Sonoma) 的 Mac - Xcode 15.0+ - Git - 10分钟初始设置

步骤1：克隆仓库（1分钟）

# 克隆主仓库
git clone https://github.com/FortressAI/FoTApple.git
cd FoTApple

# 查看结构
ls -la

仓库结构：

FoTApple/
├── apps/                    # 3个应用（Clinician、Legal、Education）
├── packages/                # 6个可重用Swift包
├── scripts/                 # 自动化脚本
└── docs/                    # 文档

步骤2：自动设置（5分钟）

对于Clinician iOS应用：

cd apps/ClinicianApp/iOS
./setup.sh

setup.sh的功能： 1. ✅ 验证系统依赖（Xcode、xcodegen） 2. ✅ 如缺少则安装xcodegen 3. ✅ 生成Xcode项目文件 4. ✅ 解析Swift包依赖 5. ✅ 创建初始配置 6. ✅ 运行基础设施验证

持续时间： 首次运行约5分钟

步骤3：构建和测试（4分钟）

在Xcode中打开：

# setup.sh之后
open ClinicianApp.xcodeproj

或使用构建脚本：

# 运行测试
./build.sh test

# 为iPhone构建
./build.sh run

# 为Mac构建
./build.sh run-mac

# 查看所有选项
./build.sh help

首次测试：

# 运行完整套件（54+测试）
./build.sh test

# 运行特定测试
xcodebuild test \
  -scheme ClinicianApp \
  -destination 'platform=iOS Simulator,name=iPhone 15'

可以离线工作吗？

离线功能：

✅ 完全离线工作： - 查看现有患者/案件/学生 - 添加新记录 - 本地推理计算（AKG GNN） - SOAP笔记、期限计算、作业
