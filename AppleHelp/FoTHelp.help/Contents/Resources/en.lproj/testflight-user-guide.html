  1. Download TestFlight (if not already installed)

Solutions: 1. Enable Siri: - Settings ‚Üí Siri & Search ‚Üí Enable 2. Enable App in Siri: - Settings ‚Üí [App Name] ‚Üí Siri & Search ‚Üí Enable 3. Re-add Shortcuts: - Open Shortcuts app - Browse FoT shortcuts - Re-add to Siri 4. Physical Device Required: - Siri integration only works on real devices, not simulator

Cryptographic Receipt Not Generating

Solutions: 1. Check VQbit engine status (console logs) 2. Ensure device has free memory (64MB+) 3. Restart app 4. Check for engine initialization errors

Data Not Syncing Across Devices

Symptoms: Entry on iPhone not appearing on iPad

Solutions: 1. Verify iCloud signed in on all devices 2. Check iCloud Drive enabled 3. Ensure all devices on same Apple ID 4. Manual sync: Pull down to refresh 5. Check network connection

Solutions: 1. Check internet connection 2. Verify RxNav API status: https://rxnav.nlm.nih.gov 3. Retry in 5 minutes (API may be temporarily down) 4. Use manual drug reference as backup

Photos/Evidence Not Capturing (Legal)
