Para Médicos

FoT Clinician - Soporte de Decisiones Médicas

Lo Que Necesitará: - iPhone (iOS 17+), Mac (macOS 14+), o Apple Watch (watchOS 10+) - 5 minutos para configurar - Conexión a Internet (para configuración inicial y base de datos de medicamentos)

Paso 1: Instalar (2 minutos)

Beta de TestFlight (Disponible Ahora): 1. Envíe un correo a beta@fortressai.com con su: - Especialidad médica - Número NPI (para verificación) - Preferencia de dispositivo (iPhone/Mac/Watch) 2. Reciba invitación de TestFlight en 24-48 horas 3. Instale la app desde el enlace de TestFlight

App Store (Próximamente Q2 2026): - Busque "FoT Clinician" - Descargue e instale - Incluye prueba gratuita de 30 días

Paso 2: Configure Su Cuenta (1 minuto)

  1. Inicie FoT Clinician
  2. Cree cuenta con Face ID o Touch ID
  3. Configure preferencias:
  4. Unidades (métricas o imperiales)
  5. Guías clínicas (ACC/AHA, GINA, etc.)
  6. Configuración de notificaciones
  7. Complete breve tutorial (omita si es experimentado)

Paso 3: Pruebe Funciones Clave (2 minutos)

Agregue Su Primer Paciente:

1. Toque el botón "+"
2. Ingrese detalles del paciente:
   - Nombre (o use "Juan Pérez" para demo)
   - Edad: 68
   - Género: Masculino
3. Agregue alergias: Penicilina
4. Agregue medicamentos actuales:
   - Metformina 1000mg VO BID
   - Lisinopril 20mg VO diario
5. Guarde paciente

Verifique Interacciones Medicamentosas:

1. Seleccione paciente "Juan Pérez"
2. Toque pestaña "Encuentro"
3. Desplácese a sección "Medicamentos"
4. Toque "Agregar Medicamento"
5. Busque "aspirina"
6. Seleccione "Aspirina 81mg"
7. Toque "Verificar Interacciones"
8. ⚠️ Vea advertencia de interacción con metformina
9. Revise mecanismo y recomendaciones

Genere Nota SOAP:

1. En el encuentro, agregue:
   - Motivo de consulta: "Dolor torácico"
   - Signos vitales: PA 145/92, FC 98, FR 18
2. Desplácese a sección "Nota SOAP"
3. Toque "Generar"
4. Revise nota profesional autogenerada
5. Toque "Exportar" para compartir (PDF)

Ver Prueba Criptográfica:

1. Al final de la nota SOAP
2. Vea "Prueba Blockchain: 01JD7X8K..."
3. Toque para ver prueba completa
4. Vea raíz Merkle, firma Ed25519
5. Verificable independientemente en SafeAICoin

Para Abogados

FoT Legal US - Gestión de Práctica Legal

Lo Que Necesitará: - iPhone (iOS 17+), iPad (iOS 17+), o Mac (macOS 14+) - 5 minutos para configurar - Su información de admisión al colegio de abogados

Paso 1: Instalar (2 minutos)

Beta de TestFlight (Disponible Ahora): 1. Envíe un correo a beta@fortressai.com con su: - Admisión al colegio de abogados estatal y año - Número de colegiado (para verificación) - Áreas de práctica 2. Reciba invitación de TestFlight 3. Instale la app

Paso 2: Configure Su Cuenta (1 minuto)

  1. Inicie FoT Legal US
  2. Cree cuenta con Face ID o Touch ID
  3. Configure preferencias:
  4. Jurisdicción primaria (federal/estatal)
  5. Áreas de práctica (civil, penal, etc.)
  6. Reglas del tribunal (FRCP, reglas locales)
  7. Complete tutorial

Paso 3: Pruebe Funciones Clave (2 minutos)

Cree Su Primer Caso:

1. Toque el botón "+"
2. Ingrese detalles del caso:
   - Nombre: Smith v. Johnson Corp.
   - Tipo: Civil - Incumplimiento de contrato
   - Tribunal: Tribunal de Distrito de EE.UU., ND California
   - Presentado: [Fecha de hoy]
3. Agregue partes:
   - Cliente: Sarah Smith
   - Oponente: Johnson Corporation
4. Guarde caso

Cálculo Automático de Plazos:

1. Abra caso "Smith v. Johnson"
2. Vea pestaña "Plazos"
3. Vea plazos calculados automáticamente:
   - Vencimiento de respuesta: 21 días (FRCP 12(a)(1)(A)(i))
   - Divulgaciones iniciales: [Fecha] (FRCP 26(a)(1)(C))
   - Conferencia de descubrimiento: [Fecha] (FRCP 26(f))
4. Cada plazo incluye cita de regla
5. Recordatorios configurados automáticamente

Genere Cita Bluebook:

1. Toque pestaña "Investigación"
2. Busque "NFIB v Sebelius"
3. Seleccione caso de resultados
4. Toque "Copiar Cita"
5. Cita copiada en formato Bluebook:
   Nat'l Fed'n of Indep. Bus. v. Sebelius, 567 U.S. 519 (2012).

Análisis Constitucional:

1. Toque pestaña "Análisis"
2. Ingrese pregunta: "¿Puede el Congreso requerir seguro médico?"
3. El sistema proporciona:
   - Disposiciones constitucionales relevantes
   - Precedente de la Corte Suprema (NFIB v. Sebelius)
   - Análisis de Cláusula de Comercio vs. Poder Tributario
   - Estrategia recomendada
4. Revise prueba blockchain del análisis

Para Desarrolladores

Construir y Contribuir

Lo Que Necesitará: - Mac con macOS 14+ (Sonoma) - Xcode 15.0+ - Git - 10 minutos para configuración inicial

Paso 1: Clonar Repositorio (1 minuto)

# Clonar el repositorio principal
git clone https://github.com/FortressAI/FoTApple.git
cd FoTApple

# Ver estructura
ls -la

Estructura del Repositorio:

FoTApple/
├── apps/                    # 3 aplicaciones (Clinician, Legal, Education)
├── packages/                # 6 paquetes Swift reutilizables
├── scripts/                 # Scripts de automatización
└── docs/                    # Documentación

Paso 2: Configuración Automatizada (5 minutos)

Para App Clinician iOS:

cd apps/ClinicianApp/iOS
./setup.sh

Lo Que Hace setup.sh: 1. ✅ Verifica dependencias del sistema (Xcode, xcodegen) 2. ✅ Instala xcodegen si falta 3. ✅ Genera archivo de proyecto Xcode 4. ✅ Resuelve dependencias de paquetes Swift 5. ✅ Crea configuración inicial 6. ✅ Ejecuta validación de infraestructura

Duración: ~5 minutos en la primera ejecución

Paso 3: Construir y Probar (4 minutos)

Abrir en Xcode:

# Después de setup.sh
open ClinicianApp.xcodeproj

O Usar Script de Construcción:

# Ejecutar pruebas
./build.sh test

# Construir para iPhone
./build.sh run

# Construir para Mac
./build.sh run-mac

# Ver todas las opciones
./build.sh help

Primeras Pruebas:

# Ejecutar suite completa (54+ pruebas)
./build.sh test

# Ejecutar pruebas específicas
xcodebuild test \
  -scheme ClinicianApp \
  -destination 'platform=iOS Simulator,name=iPhone 15'

¿Funciona sin conexión?

Funcionalidad Sin Conexión:

✅ Trabaja completamente sin conexión: - Ver pacientes/casos/estudiantes existentes - Agregar nuevos registros - Cálculos de razonamiento local (AKG GNN) - Notas SOAP, cálculos de plazos, asignaciones

⚠️ Requiere conexión para: - Verificaciones de interacciones medicamentosas (API RxNav) - Investigación de jurisprudencia (bases de datos legales) - Atestación blockchain (sumisión de pruebas) - Sincronización entre dispositivos

Datos Almacenados Localmente: - SQLite + GRDB para persistencia local - Encriptado con claves gestionadas por Secure Enclave - Sincronizable vía iCloud cuando esté conectado
